Analysis Approach

Step 1: Recursive Variable Detection


def find_variable_references(obj):
    """Recursively find all {{Variable}} patterns"""
    if isinstance(obj, str):
        # Extract variable references
        return re.findall(r'\{\{([^}]+)\}\}', obj)
    elif isinstance(obj, dict):
        # Check all dictionary values
        for value in obj.values():
            find_variable_references(value)
    elif isinstance(obj, list):
        # Check all list items
        for item in obj:
            find_variable_references(item)

Optimization Strategies

1. Safe Variable Removal Process

  1. Export container backup
  2. Remove variables with 0 references
  3. Test in Preview mode
  4. Publish with confidence

Result: 20-30% container size reduction

2. Caching Heavy Variables

Before: Re-evaluated 400+ times


function() {
    // Complex calculation every time
    const userData = parseUserData();
    const segment = calculateSegment(userData);
    return segment;
}

After: Evaluated once, cached


function() {
    if (!window._gtmCache) window._gtmCache = {};
    
    if (!window._gtmCache.userSegment) {
        const userData = parseUserData();
        window._gtmCache.userSegment = calculateSegment(userData);
    }
    
    return window._gtmCache.userSegment;
}

Performance gain: 10-50ms per page
